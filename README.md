# React TypeScript one-minute speed typing test

### Instructions

- Download or clone the repo:

```bash
git clone https://github.com/awran5/react-one-minute-speed-typing-test-ts.git
```

- Install

```bash
# npm
npm install

# Yarn
yarn install
```

### Dependencies

- none
